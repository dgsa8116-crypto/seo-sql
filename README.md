# 前導頁

教育型百家樂前導頁，內容聚焦規則說明、新手指南、下注流程、常見術語、FAQ、責任娛樂、條款與隱私政策。網站為純靜態輸出，可部署到 GitHub Pages，不需要付費後端服務。

目前版型採用都市極簡視覺風格：中性色底、單一青綠 CTA accent、響應式網格、文章 TOC、摺疊 FAQ、手機底部 CTA 與輕量 CSS。

## Tech Stack

- Plain HTML/CSS/JavaScript generated by Node.js scripts
- No runtime framework and no third-party dependencies
- GitHub Actions build, SEO update, validation, and Pages deployment
- Responsive urban minimalist UI with no client-side framework

## Local Development

```bash
npm run build
npm run validate
```

Build output is written to `dist/`. Generated SEO files are also written to `public/`.

## Update Registration URL

Edit `src/config/site.js`:

```js
registerRedirectUrl: "https://www.example.com/register"
```

For GitHub Actions deployment, set repository variable `REGISTER_REDIRECT_URL` to override the config without editing source files.

## Update SEO Content

- Page content: `src/content/pages.js`
- Keyword policy: `src/content/keywords.js`
- Site-wide defaults: `src/config/site.js`

The build script generates:

- `sitemap.xml`
- `robots.txt`
- `keyword-recommendations.json`
- `keyword-recommendations.md`
- `seo-report.md`

## GitHub Pages Deployment

Push the project to a GitHub repository and enable GitHub Pages with the source set to GitHub Actions. The workflow in `.github/workflows/pages.yml` builds the static site, updates SEO outputs, validates the result, and deploys `dist/`.

The free Pages URL uses:

- `https://USERNAME.github.io/REPOSITORY_NAME/`

## Compliance Notes

The content avoids guaranteed profit, risk-free betting claims, platform exploit language, underage targeting, fake testimonials, and hidden redirects. CTA links are visible, configurable, and use safe outbound link attributes.
